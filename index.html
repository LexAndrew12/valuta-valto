Összesített költés:

0 HUF

0 RON

Új Teljes összeg:

0 HUF

0 RON

Gombok funkciói:

  • CE: Teljes újraindítás
  • C: Aktuális bevitel törlése
  • R: RON → HUF váltás
  • H: HUF → RON váltás
  • MIN: Összeg kivonása
  • ADD: Összeg hozzáadása
  • Bevitel: Érték megerősítése
  • i: Információs panel
  • ←: Utolsó karakter törlése
